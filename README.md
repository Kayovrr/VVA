Este programa foi feito apenas para fins de teste e aprendizado. Para falar a verdade, a ideia inicial era um CRUD; no fim, virou isso aí. Com certeza está cheio de bugs e erros, mas, como diz o ditado, errar é humano(e eu sou humano demais)

<b>Para rodar, é preciso:</b>
<br>. Instalar a biblioteca brapi: pip install brapi
<br>. Instalar a biblioteca requests: pip install requests
<br>. Instalar a biblioteca matplotlib: pip install matplotlib
<br>. Rodar no interpretador Python 3.13

<b>OBS: Por limitações da API, ele apenas funcionará com as seguintes ações:</b>
<br>. PETR4 (Petrobras PN)
<br>. MGLU3 (Magazine Luiza ON)
<br>. VALE3 (Vale ON)
<br>. ITUB4 (Itaú Unibanco PN)
